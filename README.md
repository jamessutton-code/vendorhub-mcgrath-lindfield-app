# Vendor Hub App

Initial scaffold for the McGrath Lindfield Vendor Hub product.

## Stack
- Next.js
- React
- Supabase

## Routes
- `/` home
- `/admin` admin placeholder
- `/[campaignSlug]` vendor campaign route placeholder
